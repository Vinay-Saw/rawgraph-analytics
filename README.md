# Organizational Spending Analysis - Sunburst Diagram

## Project Overview
This repository contains a professional Sunburst Diagram visualization created for Ankunding LLC's organizational spending analysis. The visualization presents a hierarchical breakdown of departmental expenses across the organization.

## Visualization Details

### Chart Type
Sunburst Diagram - Created using RAWGraphs (https://rawgraphs.io/)

### Data Structure
The visualization represents organizational spending across:
- **5 Departments:** Engineering, Marketing, Operations, Sales, Finance, HR
- **Multiple Expense Categories:** Personnel, Software, Hardware, Training, Advertising, Events, Facilities, Equipment, Travel, etc.
- **Total Budget Analysis:** Approximately $1.5M across all departments

### Key Insights
1. **Engineering** has the highest overall spending, driven by personnel and software costs
2. **Marketing** shows significant investment in advertising and digital channels
3. **Personnel costs** represent the largest expense category across all departments
4. **Operations** maintains balanced spending across facilities, equipment, and utilities

### Hierarchy Structure
```
Level 1: Department (outer ring)
Level 2: Expense Type (inner segments)
Size: Amount (segment proportions)
Color: Department-based color coding
```
